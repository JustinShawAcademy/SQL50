## 1943. [Confirmation Rate](https://leetcode.com/problems/confirmation-rate/?envType=study-plan-v2&envId=top-sql-50)
* Medium Problem

### SQL Solution
#### Concepts / Syntax
`CASE WHEN`, `LEFT JOIN`, `GROUP BY`, `AVG()`

#### Intuition
For this problem, we have to `LEFT JOIN` `Confirmations` onto `Signups` so that we have all the rows of sign ups that did or did not request for a confirmation. We can then `GROUP BY` the people that signup up and `AVG()` the amount of times they selected `action = confirmation`.

#### Approach
1. We want to `LEFT JOIN` `Confirmation` onto `Signup` so that we have all the rows of sign ups that did or did not request for a confirmation.
```sql
SELECT *
FROM Signups s
LEFT JOIN Confirmations c
ON s.user_id = c.user_id
```

2. We want to `GROUP BY` s.user_id as we want the start performing aggreation on the action column.

```sql
SELECT *
FROM Signups s
LEFT JOIN Confirmations c
ON s.user_id = c.user_id
GROUP BY s.user_id
```

```
| user_id | time_stamp          | user_id | time_stamp          | action    |
| ------- | ------------------- | ------- | ------------------- | --------- |
| 3       | 2020-03-21 10:16:13 | 3       | 2021-07-14 14:00:00 | timeout   |

| 2       | 2020-07-29 23:09:44 | 2       | 2021-02-28 23:59:59 | timeout   |
| 6       | 2020-12-09 10:39:37 | null    | null                | null      |
```

3. Now for tricky part, within each aciton pile, we want to find the sum of `'confirmed' / ('confirmed' + 'timeout')`. This is done with `CASE WHEN` followed by `AVG()`.
```sql
SELECT 
    s.user_id
    , ROUND(AVG(CASE WHEN c.action = 'confirmed' THEN 1 ELSE 0 END), 2) AS confirmation_rate
FROM Signups s
LEFT JOIN Confirmations c
ON s.user_id = c.user_id
GROUP BY s.user_id
```

```
| ------- | ----------------- |
| 3       | 0                 |
| 7       | 1                 |
| 2       | 0.5               |
| 6       | 0                 |
```


#### Code
```sql
SELECT 
    s.user_id
    , ROUND(AVG(CASE WHEN c.action = 'confirmed' THEN 1 ELSE 0 END), 2) AS confirmation_rate
FROM Signups s
LEFT JOIN Confirmations c
ON s.user_id = c.user_id
GROUP BY s.user_id
```
